# Quiz Project Demo

这是一个基于 Vue 3 和 Node.js 的在线答题系统。

## 项目特点
- **前后端分离**：前端使用 Vue 3 + Ant Design Vue，后端使用 Node.js + SQL Server。
- **AI 智能批改**：集成智谱清言 GLM-4 模型，支持问答题自动评分。
- **模块化设计**：支持单选、多选、判断、问答四种题型。
- **数据安全**：所有接口强制 Token 校验。

## 技术栈
- 前端：Vue 3, Vite, Ant Design Vue 4.x
- 后端：Node.js, Express, SQL Server (mssql)
- AI：智谱清言 GLM-4-flash
